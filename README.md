# Simple-HIDS-C-C-
İng = A lightweight C-based Host Intrusion Detection System (HIDS) designed for monitoring system calls and file integrity on Linux systems.

Tr = C ile geliştirilen, Linux sistemlerde dosya ve işlem hareketlerini izleyerek şüpheli aktiviteleri tespit etmeyi amaçlayan bir Host-based Intrusion Detection System (HIDS) projesi.


Bu projede, sistemlerin güvenliğini en alt seviyeden takip edebilmek amacıyla basit bir Host-based Intrusion Detection System (HIDS) geliştiriyorum. Bu sistemle; sistem çağrılarının (syscalls) izlenmesini, dosya bütünlüğünün kontrolünü, bellek yönetimini ve sistem loglarının anlık takibini yapmayı planlıyorum.

🚀 Geliştirme Yaklaşımı ve Motivasyon

Bu proje tamamen "öğrenerek inşa etme" prensibiyle ilerlemektedir. Siber güvenlik ve sistem programlama temellerini daha iyi kavramak adına, projeyi bir kez tamamen baştan yazarak mimariyi sadeleştirdim ve performans optimizasyonuna odaklandım. Amacım, sadece hazır araçları kullanmak değil, bu araçların arka planda nasıl çalıştığını derinlemesine anlamaktır.
